# CardGame

Android card game built with Kotlin and Jetpack Compose.

## Features

- Modern Android architecture
- Jetpack Compose UI
- Playing card deck with SVG assets
- Multiple card game implementations
- Responsive layouts for different screen sizes

## Tech Stack

- Kotlin
- Jetpack Compose
- Android ViewModel
- Coroutines
- Flow
- Material 3
